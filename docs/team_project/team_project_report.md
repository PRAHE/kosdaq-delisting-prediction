# 한국 코스닥 상장폐지 조기경보 — 팀 프로젝트 보고서 (골격)

> 발표·보고서 초안. 각 섹션은 슬라이드 1장 분량으로 확장 가능.

---

## 핵심 한 줄
> **"좋은 알고리즘보다 좋은 정보가 중요했다."** 재무비율만으로 PR-AUC 0.29에
> 머물던 모델이, **감사의견 데이터를 통합**하자 0.37로 올랐고, 이를 **위험등급·
> 위험기업 watchlist** 형태의 실사용 서비스로 구성했다. 복잡한 딥러닝 없이 단순·
> 해석가능한 모델로 달성.

---

## 1. 문제 정의
- **목표:** DART 재무제표로 한국 코스닥 기업의 **상장폐지를 1년 전에 예측**(조기경보).
- **데이터:** 1,536종목 분기 패널. 학습 2015–2022 / 검증 2023 / **평가 2024**.
- **난점:** 극심한 불균형 — 2024년 평가셋에서 실제 상폐는 **5,311건 중 56건(약 1%)**.
  이 희소성 때문에 "정확도"는 무의미하고 **PR-AUC**와 **상위 K개 정밀도**로 평가.

## 2. 접근 — 모델보다 데이터
초기 9개 개선 실험(부스팅·튜닝·SMOTE 등)이 baseline을 넘지 못했다. 원인 분석 결과
**알고리즘 교체로는 성능이 거의 오르지 않았고**, 진짜 차이는 **재무제표 바깥의
정보**에서 나왔다. → 전략을 "더 좋은 모델 찾기"에서 **"더 좋은 정보 통합하기"**로 전환.

## 3. 모델 발전 — 감사의견 통합이 핵심
DART 감사의견(적정/한정/의견거절)을 시점 누수 없이(직전연도 의견만) 통합:

| 단계 | 사용 정보 | PR-AUC (2024) | P@20 |
|------|-----------|---------------|------|
| 1차 | 재무비율만 (33) | **0.29** | 0.60 |
| 2차 | + **감사의견** (38) | **0.37** | 0.60 |
| 운영 모델 | + 조기 관리종목 (44) | **0.39** | **0.75** |

- 감사의견 한 종류 추가로 **+0.08** — 단일 개선 중 최대. (참고: 비적정 2년 연속
  기업의 실제 상폐율은 38.5%로 평균의 96배.)
- **알고리즘은 무관:** RF·로지스틱회귀가 통계적으로 동급. → 해석이 쉬운 단순 모델 채택.

## 4. 결과 — 위험등급 시스템
예측 확률을 4단계 등급으로 변환(임계치는 2023 검증셋 기준, 2024에 적용). **처음 보는
2024 데이터에서 등급별 실제 상폐율이 단조로 증가** → 등급이 신뢰할 만함을 입증:

| 등급 | 의미 | 실현 상폐율 | 상폐기업 탐지 비중 |
|------|------|------------|---------------------|
| **D** | 상폐위험 | **31.1%** | 57% |
| C | 고위험 | 2.7% | 11% |
| B | 주의 | 1.4% | 21% |
| A | 정상 | 0.15% | 11% |

기저 상폐율 1%의 **D등급 ≈ 30배**. (모델 확률이 잘 보정돼 등급화 신뢰 가능 — ECE(Expected Calibration Error) 0.003.)

감사의견·재무지표로 자동 설명**되어 그대로 대시보드에 노출 가능.

## 5. 활용 시나리오
- **위험등급 대시보드:** 기업명 · 위험도 · 등급(A~D) · 주요원인 · 감사의견.
- **투자자/규제 경보:** D·C 등급 기업 자동 알림, 상위 위험기업 TOP20 리스트.
- **모니터링:** 분기마다 등급 갱신, 등급 상승(예: B→D) 기업 추적.

## 6. 한계 & 정직성 (의도적 설계 결정)
- **감사의견은 절반만 잡는다.** 상폐 기업의 약 48%만 사전에 비적정 의견을 받았고,
  나머지 52%는 적정 의견인데도 상폐했다 → 이들은 **재무지표로 보완**(주요원인 칼럼이
  감사/재무 두 갈래인 이유).
- **주가·거래량 피처는 일부러 뺐다.** 넣으면 지표(ROC 0.99)가 더 좋아 *보였지만*,
  분석 결과 그 성능의 대부분은 **이미 거래정지된 "죽은 주식"을 골라낸 것**이었다.
  상폐를 *미리* 경고하는 게 목적이므로 사후 상태 탐지는 제외 — **성능 숫자보다 제품
  목적에 맞는 정직한 선택.**
- **검증 범위:** 모든 결과는 2024년 평가 기준(in-sample). 실서비스 신뢰를 위해
  2025년 데이터로 재검증 예정.
