## 초록 (Abstract)

#1·#3 대조 실험에서 천장 돌파(PR-AUC 0.26→0.41)의 1차 동력이 **시계열이 아니라
외부 부실 신호(감사·시장)** 임이 확정됐다. 그러자 자연스러운 반론이 둘 생긴다 —
(i) 감사의견은 KRX 38조의 상폐 트리거 그 자체이니 **누수(leakage)** 아닌가?
(ii) 그렇다면 감사 피처 하나로 문제가 **자명하게(tautology)** 풀리는 것 아닌가?
본 검증이 둘 다 정량적으로 반박한다.

**#4 (누수 검증):** test 양성 56개 전부에서 사용된 감사의견의 회계연도(year−1)는
상폐연도(year+1)보다 **정확히 2 회계연도 앞선다**(gap 분포 = {2년: 56}). 즉
모델이 보는 의견은 항상 상폐 사건 이전에 공개된 정보다 — **전향적 누수가 아니다**.

**#2 (자명성 검증):** 감사 5피처 **단독** RF의 test PR-AUC는 **0.252**로 재무
33피처 단독(RF-33, 0.258)과 사실상 동일하다. 단일 피처 트리비얼 룰은 더 낮다
(`audit_nonclean_consec` 0.227, `audit_opinion_t` 0.149). **감사는 단독으로는
천장을 못 깬다.** 재무와 결합(RF+audit, 0.367)했을 때 비로소 +0.11이 나온다 —
즉 감사의 가치는 **상호작용**(재무상 애매한 기업 중 누가 실제 상폐하는지 판별)
이지 자명한 트리거가 아니다.

**핵심 부수 발견:** 양성 56개 중 **27개(48%)만 이미 비적정(의견거절) 의견**을
가졌고, **29개(52%)는 적정 의견인데도 상폐**했다. 후자가 S0 진단이 지목한
"재무·감사상 정상으로 보이는 confident FN"의 실체다 — 감사로는 원리적으로 못
잡으며, 재무 궤적·시장 신호가 필요한 영역.

## 1. #4 — Leakage audit

감사 피처는 `audit_features.py` 구조상 `target_year − 1 … −5` 의견만 사용한다
(피처 산출 단계의 전향적 누수 없음). 본 검증은 **실제 양성 케이스**에서 그 시점
정합성을 확인한다.

fixed_N1 양성(delta=1)은 정의상 `delist_year = year + 1`. 사용되는 의견은
`audit_opinion_t = year − 1` 회계연도. 따라서:

```
gap = delist_year − opinion_fiscal_year = (year+1) − (year−1) = 2 회계연도
```

| 검증 항목 | 결과 |
|-----------|------|
| 양성 수 | 56 |
| 의견→상폐 gap 분포 | **{2년: 56}** (전원 2년 앞섬) |
| `audit_observed = 1` | 56/56 (100%) |
| Y−1 의견이 이미 비적정 | **27/56 (48%)** — 자명한 규제 트리거 |
| Y−1 의견이 적정인데 상폐 | **29/56 (52%)** — 감사-blind 케이스 |
| 양성의 `audit_opinion_t` 분포 | 적정(0): 29, 의견거절(3): 27 (이분화) |

> 전체 표: `results/research_audit_check/audit_leakage_table_positives.csv`

**결론:** 의견은 상폐보다 항상 2년 앞서므로 누수가 아니다. 양성의 절반만 감사가
사전 포착하며, 의견거절은 정당한 2년-선행 경고다.

## 2. #2 — Audit-only 단독 모델

| 모델 (test 2024) | PR-AUC | ROC-AUC | P@20 |
|------------------|--------|---------|------|
| RF, 감사 5피처만 | **0.252** | 0.717 | 0.55 |
| trivial: rank by `audit_nonclean_consec` | 0.227 | — | — |
| trivial: rank by `audit_opinion_t` | 0.149 | — | — |
| trivial: rank by `audit_nonclean_5y` | 0.200 | — | — |
| (참고) RF-33 재무만 [#1] | 0.258 | 0.842 | 0.50 |
| (참고) RF+audit 38 [#1] | 0.367 | 0.870 | 0.60 |
| (참고) RF-44 [#1] | 0.413 | 0.982 | 0.55 |

**관찰:** 감사 단독(0.252) ≈ 재무 단독(0.258). 어느 쪽도 단독으로는 결합
(0.367)에 못 미친다. 감사 피처의 +0.11 기여는 **재무 신호와의 상호작용**에서
나온다 — 감사가 "재무상 애매한 기업 중 의견거절을 받은 쪽"을 위로 끌어올린다.

이는 누수/자명성 우려를 동시에 해소한다: 의견거절이 곧 상폐라면 audit-only가
0.9+를 찍어야 하지만 실제로는 0.25에 그친다. 양성의 48%만 비적정이고, 비적정
기업이 모두 상폐하지도 않기 때문이다(A1 §2.5의 38.5% < 100%와 정합).

## 4. 남은 점검 과제와의 연결

- 양성의 52%가 감사-blind(적정 의견인데 상폐) → 이들을 무엇이 잡는가? #1에서
  스냅샷 `RF+market`의 ROC가 0.988까지 치솟은 점(주가 붕괴 = 동시점 distress)이
  유력 후보. **#8 (시장 데이터 양성-클래스 coverage / halt 종목 누수 점검)** 이
  다음 우선 검증 대상.
- A1이 "시장 미약(+0.008)"이라 한 것과 스냅샷 RF에서 시장이 지배적인 것의
  충돌도 #8에서 함께 규명.
